Map与WeakMap

Map

简述

Map对象用用于保存键值，对任何值(对象或者原始值) 都可以作为一个键或一个值。与Set相似，Map相同Key只能出现一次。

由于Map与Set用法基本相同，有需要的小伙伴可以参考Set与WeakSet那一章进行参考。接下来只列出一些常见的API，将不再具体列出相关使用用例。

属性和方法

属性

  • size：返回该集合的成员数量

方法

  • 操作方法
    • set(key, value)：向集合添加一个新的成员
    • get(key)：返回集合中的该键对应的值
    • delete(key)：从集合中移除一个成员
    • has(key)：如果值在集合中存在，返回true,否则false
    • clear(): 移除集合里所有的项
  • 遍历方法
    • keys()：返回一个包含集合中所有键的迭代器对象
    • values()：返回一个包含集合中所有值的迭代器对象
    • entries()：返回一个包含集合中所有键值对的迭代器对象
    • forEach(callback([value][,key][,map])[, thisArg])：用于对集合成员执行遍历，没有返回值

WeakMap

简述

与WeakSet特性相似，具体使用可以参考Set与WeakSet那一章。

属性和方法

属性

无属性

方法

  • set(key, value)：向集合添加一个新的成员
  • get(key)：返回集合中的该键对应的值
  • delete(key)：从集合中移除一个成员
  • has(key)：如果值在集合中存在，返回true,否则false
