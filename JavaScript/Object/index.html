对象属性的描述符

对象的每个属性都有各自的描述符可以进行配置。

  • value: ‘lihua’ # 属性值
  • writable: true # 属性是否只读
  • enumerable: true # 属性是否可被遍历
  • configurable: true # 属性能否被删除，特性能否被重新配置

可以使用Object.defineProperty进行配置。

禁止添加属性

使用Object.preventExtensions对对象进行配置，使之失去扩展能力。

封闭对象

使用Object.seal封闭一个对象。

  • 不能添加新属性
  • 已有属性不可配置（configurable被置为false）
  • 已有属性若之前可被更改，则现在仍可被更改

冻结对象

使用Object.freeze封闭一个对象。

freeze相较于seal，多了一个把writable置为false的操作，也就是对象的属性不允许被修改。

访问器属性

对象属性还拥有getter与setter两个访问器属性，可以对属性的赋值与获取进行拦截

Proxy代理

JSON对象

JSON对象API

  • JSON.stringify(value[, replacer [, space]]) // 将对象格式化为JSON格式字符串
    • value: 对象
    • replacer: 转换处理相关，详情查看MDN
    • space: 缩进的空格数，用于美化
  • JSON.parse(text[, reviver]) // 将字符串转换为对象类型
    • text: 要被转换的字符串
    • reviver: 转换器
